# Documentation archive

This directory contains superseded assessments, completed implementation plans, sprint reports,
and historical diagnostic output. The files are retained for context and provenance, not as current
project direction.

Documents are grouped by the year in which they described the project. Some internal links and line
references are intentionally historical and may no longer resolve against the current tree.

- `2024/`: early external codebase review.
- `2025/`: architecture assessments, implementation plans, sprint reports, and tooling diagnostics.
- `2026/`: the June cleanup review and its supporting audits/profiling report.

For current guidance, use [the documentation index](../README.md), [status](../STATUS.md), and
[roadmap](../ROADMAP.md).
